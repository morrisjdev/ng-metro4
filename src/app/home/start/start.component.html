Quick Start

Install

To install ng-metro4 in your angular application just run:
ng add ng-metro4

To generate a first layout run:
ng g ng-metro4:layout
This will overwrite your app.component.html so be careful if you already made some changes
If you haven't created an application yet you can create one using:
npm install -g @angular/cli
and
ng new <AppName>

You can then start using the application:
ng serve

For more information see: Angular setup

You are now ready to use ng-metro4.
You can start browsing the components to get an image of what is possible.

Manual setup

If you don't want to use the method above you can also install all dependencies by yourself.

Add dependencies

To install ng-metro4 in your angular application first execute this command:

npm install -S ng-metro4 metro4 linq4js
