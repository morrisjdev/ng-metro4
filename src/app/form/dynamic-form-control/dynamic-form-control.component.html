Dynamic form control

The m4-dynamic-form-control is a component that renders a form control dynamically. You can completely control its behavior and appearance on the code side. This enables easy development of complex code driven forms.
]]>
;\n\n constructor() {\n \tthis.formControl = new M4FormControl(InputComponent);\n } ]]>
You can use any available control type as a control for the dynamic form control. These are the controls available:
  • Input
  • Material Input
  • Textarea
  • Checkbox
  • Switch
  • Radio
  • Calendar
  • Calendar Picker
  • Date Picker
  • Time Picker
  • Select
  • Tag Input
  • Slider
  • Spinner
  • Rating
  • Keypad
  • File
